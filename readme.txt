Add google maps to your site

== Description ==

Allows you to use short codes and a editor to add google maps to your site. Documentation shown on the My Locations admin page.


== Installation ==

1) Install WordPress 3.5.1 or higher

2) Download the following files:
http://downloads.wordpress.org/plugin/my-locations-1.0.zip

3) Login to WordPress admin, click on Plugins / Add New / Upload, then upload the zip file you just downloaded.

4) Activate the plugin.
